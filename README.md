# CodeAlpha_JobBoard

A RESTful Job Board API built with Node.js, Express.js, and MongoDB.

Developed as part of the Backend Development Internship at CodeAlpha (June 2026).

## Features

- Employer and candidate account management
- Job listing creation, search, and filtering
- Resume upload and management
- Job application submission and tracking
- Application status updates and employer notifications

## Tech Stack

- **Runtime:** Node.js
- **Framework:** Express.js
- **Database:** MongoDB with Mongoose
- **Architecture:** MVC (Model-View-Controller)

## Getting Started

```bash
npm install
cp .env.example .env
# Fill in your environment variables
npm run dev
```
